# SEO checklist (practical)

## On-page essentials (per page)

- **Search intent**: define the query + what a searcher expects (guide vs product vs comparison).
- **Title tag**: unique, includes primary keyword + value prop + brand where sensible.
- **Meta description**: present and compelling (not a ranking factor, but drives CTR).
- **H1**: exactly one primary H1; matches title intent.
- **Headings**: H2/H3 structure that mirrors buyer questions.
- **Image alt text**: describe product/function (accessibility + minor SEO).
- **Internal links**: link to key categories, compatible accessories, guides.
- **Schema**:
  - Product (price/availability/brand)
  - Organization / LocalBusiness (if relevant)
  - FAQ (only for genuine FAQs)
- **Canonical**: correct canonical on variants and paginated pages.

## Technical

- Robots: no accidental noindex.
- XML sitemap: exists + submitted (GSC).
- 404s / redirects: avoid chains.
- Core Web Vitals: address LCP/INP/CLS.
- Mobile usability: tap targets, font size, layout.

## Content strategy (ecommerce)

- Create **collections pages** that explain the category (not just product grids).
- Create **comparison pages** (portable vs wallbox, solar-only vs scheduled charging).
- Create **support content**: installation, compliance, rebates, compatibility.
